.. _dpm_component_run:

dpm component run
-----------------

pull down and run a remote component

Synopsis
~~~~~~~~


pull down and run a remote component.
The registry to pull from is the one configured via $DPM_HOME $DPM_REGISTRY/dpm-config.yaml

::

  dpm component run <component name> <component version> [args] [flags]

Options
~~~~~~~

::

  -h, --help   help for run
