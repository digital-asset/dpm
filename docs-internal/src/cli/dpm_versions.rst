dpm versions
------------

show sdk versions

Synopsis
~~~~~~~~


show sdk versions

::

  dpm versions [flags]

Options
~~~~~~~

::

  -s, --active          display the active sdk version only
  -A, --all             display remote versions
  -h, --help            help for versions
  -o, --output string   output format: json, table (default "table")
