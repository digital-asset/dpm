dpm component init
------------------

initialize a component in the current directory

Synopsis
~~~~~~~~


initialize a component in the current directory

::

  dpm component init [flags]

Options
~~~~~~~

::

      --force   overwrite existing component.yaml and dpm.local.yaml
  -h, --help    help for init
